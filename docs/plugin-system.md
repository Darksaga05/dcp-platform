# DCP Plugin System

## Overview

DCP uses a modular plugin architecture.

Each communication protocol is implemented as an independent plugin.

## Planned Plugins

- DMR
- M17
- YSF
- FCS
- D-STAR
- P25
- NXDN
- TETRA

## Plugin Requirements

- Independent
- Hot Load Support
- Version Control
- Secure Communication
- Digital Signature Verification
