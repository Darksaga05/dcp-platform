# DCP Login Screen

## Overview

The Login Screen is the main entry point of the DCP application.

## Components

- Company Logo
- Welcome Text
- Email Input
- Password Input
- Remember Me
- Sign In Button
- Forgot Password
- Language Selector
- Theme Switcher

## Security

- Biometric Login
- Two-Factor Authentication (2FA)
- Secure Session Management

## Future Features

- Passkey Authentication
- QR Code Login
- Organization SSO
