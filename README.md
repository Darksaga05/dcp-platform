# DCP - Digital Communication Platform

## 🚀 Vision

DCP (Digital Communication Platform), amatör telsiz, profesyonel haberleşme ve acil durum iletişimini tek bir platform altında birleştirmeyi amaçlayan yeni nesil, açık ve modüler bir iletişim sistemidir.

## 🎯 Project Goals

- Digital Mobile Radio (DMR)
- M17
- D-STAR
- YSF / FCS
- P25
- NXDN
- Offline (internet olmadan) çalışma desteği
- Push-To-Talk (PTT)
- Yapay zekâ destekli ses işleme
- Çoklu platform desteği (Android, iOS, Windows, Linux, macOS)
